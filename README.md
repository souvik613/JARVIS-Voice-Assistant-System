
# 🤖 JARVIS Voice Assistant System

JARVIS is a simple Python-based voice assistant that can interact with the user using voice commands. It can perform tasks such as opening applications, searching on Google or Wikipedia, playing music, telling jokes, and basic conversations.

This project uses **speech recognition** and **text-to-speech (TTS)** to create a hands-free assistant experience similar to Iron Man's JARVIS.

---

## 🛠 Features

* Greet the user based on the time of day
* Recognize voice commands using Google Speech Recognition
* Speak responses using pyttsx3
* Tell the current time and date
* Search information on Wikipedia
* Open websites like Google, Facebook, and YouTube
* Play random music from a folder
* Open system applications such as Calculator, Notepad, and CMD
* Tell jokes and respond to basic small talk
* Exit using a voice command

---

## 💻 Requirements

* Python 3.11 or higher

---

## ▶ How to Run

1. Create a virtual environment

```bash
conda create -n jarvis python=3.11 -y
```

2. Activate the virtual environment

```bash
conda activate jarvis
```

3. Install required packages

```bash
pip install -r requirements.txt
```

4. Run the program

```bash
python jarvis.py
```
